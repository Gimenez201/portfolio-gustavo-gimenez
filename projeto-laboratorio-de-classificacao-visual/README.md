# 🧠 Análise de Viés em Classificação Visual com IA

> Projeto acadêmico desenvolvido na disciplina de **[NOME DA DISCIPLINA AQUI]** utilizando Inteligência Artificial e Visão Computacional para demonstrar os impactos do viés em datasets de treinamento.

---

## 📝 Descrição do Projeto

Este projeto explora o impacto do viés de dados em sistemas de Inteligência Artificial utilizando a plataforma Teachable Machine, do Google. O objetivo principal consiste em demonstrar como conjuntos de dados enviesados influenciam diretamente o comportamento de modelos de classificação visual, produzindo decisões incorretas e potencialmente discriminatórias.

Durante o experimento, foram criadas duas categorias de classificação:

- **Perfil Liderança**
- **Perfil Operacional**

O treinamento do modelo utilizou imagens com padrões estereotipados de aparência e vestimenta, simulando um cenário de coleta de dados não representativo. Após o treinamento, o sistema foi testado com indivíduos fora dos padrões utilizados no dataset, evidenciando falhas de inferência causadas pelo viés introduzido no processo.

O projeto foi desenvolvido como parte das atividades acadêmicas do curso de **[SEU CURSO AQUI]** na instituição **[NOME DA INSTITUIÇÃO AQUI]**.

## 🧪 Metodologia Aplicada

### 1️⃣ Definição das Classes

Foram criadas duas categorias distintas:

- Perfil Liderança
- Perfil Operacional

---

### 2️⃣ Criação do Dataset

O treinamento utilizou aproximadamente 20 imagens para cada classe, empregando critérios visuais estereotipados para simular um conjunto de dados enviesado.

---

### 3️⃣ Treinamento do Modelo

O modelo foi treinado utilizando classificação de imagens em tempo real através da plataforma Teachable Machine.

---

### 4️⃣ Testes de Inferência

Após o treinamento, pessoas fora do padrão utilizado no dataset foram analisadas pela câmera, permitindo observar erros de classificação causados pelo viés dos dados.

## ⚠️ Impacto Ético Observado

O experimento evidencia que sistemas treinados com dados limitados ou estereotipados passam a reproduzir padrões discriminatórios, criando interpretações distorcidas da realidade.

Além do erro técnico, classificações enviesadas podem gerar:

- exclusão social;
- marginalização profissional;
- reforço de preconceitos;
- invisibilidade de determinados grupos.

A atividade também demonstra a importância da supervisão humana no processo de curadoria de dados e validação de modelos de Inteligência Artificial.

---

## 🧩 Solução Mitigadora Proposta

Uma abordagem de *Human-in-the-loop* pode reduzir significativamente os efeitos do viés algorítmico. Nesse modelo, especialistas humanos participam da seleção, revisão e validação dos dados utilizados no treinamento da IA, garantindo maior diversidade, representatividade e equilíbrio entre as categorias analisadas.
